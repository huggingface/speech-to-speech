# ModelVerse Integration for speech-to-speech

[ModelVerse](https://modelverse.cn) is an AI model aggregation platform fully compatible
with the OpenAI API format. You can access DeepSeek, GPT-4, Claude and more models
through a single unified endpoint.

## Quick Start

**Step 1**: Get your API key from [ModelVerse Platform](https://modelverse.cn)

**Step 2**: Install the OpenAI SDK (if not already installed)

```bash
pip install openai
```

**Step 3**: Use ModelVerse in this project

```python
from openai import OpenAI

client = OpenAI(
    api_key="your-modelverse-api-key",
    base_url="https://api.modelverse.cn/v1",
)

response = client.chat.completions.create(
    model="deepseek-ai/DeepSeek-V3",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(response.choices[0].message.content)
```
